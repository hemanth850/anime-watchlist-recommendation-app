# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-02-17

### Added
- Monorepo setup with React web app, Express API, and shared contracts package.
- Authentication flows with signup, login, session token, and protected route checks.
- Watchlist CRUD with status transitions, ratings, notes, and episode progress.
- Personalized recommendation engine with explainability reasons.
- Catalog discovery endpoint with search/filter/sort support.
- Phase 6 release baseline:
  - Unit/integration/e2e API tests
  - GitHub Actions CI pipeline
  - Docker deployment configuration
  - Release and deployment documentation
